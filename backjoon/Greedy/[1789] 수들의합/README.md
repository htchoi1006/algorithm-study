### 문제

서로 다른 N개의 자연수의 합이 S라고 한다. S를 알 때, 자연수 N의 최댓값은 얼마일까?




---

### 입력

첫째 줄에 자연수 S(1 ≤ S ≤ 4,294,967,295)가 주어진다.



---

### 출력

첫째 줄에 자연수 N의 최댓값을 출력한다.


---

### 예제 입력

| 예제 입력1 | 예제 출력1 |
| :--- | :--- |
|  200  |   19   |




---

### 문제 접근

- 1부터 n까지 차례대로 더해 입력받은 수보다 커지게 되면 그 개수에서 1을 뺀다.
